# Glossary (quick)

- **Bounded Context**: a domain boundary with its own language and model.
- **CQRS**: separate command (write) and query (read) models.
- **Outbox**: publish events reliably after DB commit.
- **SLO**: service level objective (target reliability).
- **RED**: rate, errors, duration (request-centric metrics).
- **USE**: utilization, saturation, errors (resource-centric metrics).
